Diagrams

flowchart TB
  Tray["Menu Bar Tray"] --> Rust["Rust / Tauri Core"]
  Settings["Native Settings Window"] --> React["React + TypeScript UI"]
  React --> Rust
  Rust --> State["Persisted App State"]
  Rust --> Shim["macOS Shim / AppKit bridge"]
  Rust --> Overlay["Transparent Overlay Window"]
  React --> Renderer["Overlay Renderer"]
  Renderer --> Overlay
  Login["Launch at Login"] --> Rust

Rust owns the app shell, state, display selection, and macOS bridges. React renders settings and overlay content.

flowchart TB
  Open["Open Airmark"] --> Pick["Pick display"]
  Pick --> Mode{"Watermark mode"}
  Mode --> Text["Text + opacity + size + spacing"]
  Mode --> Image["Drop, choose, or paste image"]
  Text --> Save["Persist settings"]
  Image --> Save
  Save --> Overlay["Transparent overlay on selected display"]
  Overlay --> Work["Keyboard, mouse, and menu bar stay usable"]

The overlay is targeted to one display and remains non-blocking for normal desktop work.
